### _Cheminement de l'utilisateur_

Dans la page d'accueil il y aura un logo + navigation donc (Accueil, Biographie, Galerie, Paroles et Contact) les liens seront l'un à côté de l'autre.
En dessous une grande photos du groupe avec le nom du groupe écrit dessus.

Un aside 'News' aside est un contenu mis à part du contenu dans lequel il est placé, j'écrirais les titres du news, photos, une petite description et la date (quand est-il publier sur le site.

_scrolle vers le bas_ 

  On trouvera une section avec le titre 'agenda des concert' avec la date, le nom et le lieu. 

_scrolle vers le bas_ 
 
  On trouvera une section 'photos' que je mettrais quelque photos avec un bouton 'Voir tous' qui amènera vers la page Galerie.
 
_scrolle vers le bas_ 
  
  On trouvera une section 'Contact' avec une phrase écrit + un bouton écrit 'Formulaire' qui amènera vers la page de Contact.
  
_scrolle vers le bas_
  
  Le footer avec tous les liens réseaux sociaux (Facebook, Instagram) Apps (Bandsintown, Streaming, Spotify, Deezer, AppleMusic, Amazon, Google Play, SoundCloud)

#### Menu

Quand nous cliquons sur le lien _Biographie_ ça amènera vers la page biographie avec (une grande photo du groupe, une description, année de création, pays, genre musical, suivre l'artiste donc les liens, une photo de chaque membres du groupe avec leurs noms et leurs rôles.

Quand nous cliquons sur le lien _Galerie_ ça amènera vers la page galerie ou il y aura des photos et des vidéos de leurs concerts.

Quand nous cliquons sur le lien _Paroles_ ça amènera vers la page paroles avec tous leurs chansons (titres, paroles 'ceux qui sont écrits les paroles' citez les noms et composition & arrangements).

Quand nous cliquons sue le lien _Contact_ ça amènera vers la page contact avec une formulaire (Nom, prénom, e-mail, un message) et un bouton 'Envoyez' et tous les liens réseaux sociaux (Facebook, Instagram) Apps (Bandsintown, Streaming, Spotify, Deezer, AppleMusic, Amazon, Google Play, SoundCloud).
